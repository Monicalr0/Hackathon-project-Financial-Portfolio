About


Introduction

Our Portfolio Management System is an efficient gateway to diversified investment management. It connects with the worldwide spectrum of securities which include stocks, bonds, ETFs, mutual funds, etc. Our platform enables users to manage their portfolios with efficient tools and data visualizations. Each user would have their own user ID to keep their account manageable and private. The user interface design clearly displays the portfolio’s performance and the user’s financial information such as cash flow and total earnings. Our system helps the user improve financial visions which is capable of not only minimizing risk exposure but also optimizing decision making.

Key Features

  • Buy/Sell Functionality: The user can add and remove different types of securities in their portfolio by selecting the buy/sell radio buttons and entering the ticker symbol and the number of shares on the Transactions page. The system automatically updates the portfolio’s performance and the user’s financial information after each transaction.

  • Portfolio Tracking: Performance and information data for all the securities’ is tracked in the system, which are Total Return, Net Gain/Loss, Market Value, 52-week High, 52-week Low, and Asset type. The strong presentation of information enables the users to keep track of their as the number of shares held and buy-in the price of their securities.

  • Performance tracking: Users are able to see the trend of the security and the rate of returns with effective graphic presentations and numeric values.

  • Diversification Analysis: The platform tracked different market sectors and product types.

  • Transaction Management: Users are able to track transaction history as the portfolio has an aggregate transaction table that records the number of shares and the sales/buy price of each transaction.

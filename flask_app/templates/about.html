About


Introduction

Our Portfolio Management System serves as a highly effective gateway to comprehensive investment management. It establishes connections within the global array of securities, encompassing stocks, bonds, ETFs, mutual funds, and more. Through our platform, users gain the capability to adeptly oversee their portfolios, utilizing a suite of efficient tools and data visualizations. The user interface has been meticulously designed to present the portfolio's performance transparently, alongside the user's financial details, including cash flow and total earnings. Our system's functionality goes beyond risk mitigation, offering users the potential to optimize decision-making and enhance financial foresight.

Key Features

  • Buy/Sell Functionality: The user can add and remove different types of securities in their portfolio by selecting the buy/sell radio buttons and entering the ticker symbol and the number of shares on the Transactions page. The system automatically updates the portfolio’s performance and the user’s financial information after each transaction.

  • Portfolio Tracking: Performance and information data for all the securities’ is tracked in the system, which are Total Return, Net Gain/Loss, Market Value, 52-week High, 52-week Low, and Asset type. The strong presentation of information enables the users to keep track of their as the number of shares held and buy-in the price of their securities.

  • Performance tracking: Users are able to see the trend of the security and the rate of returns with effective graphic presentations and numeric values.

  • Diversification Analysis: The platform tracks different market sectors and product types.

  • Transaction Management: Users are able to view their transaction history since the portfolio has an aggregate transaction table that records the number of shares and the sale/purchase price of each transaction.

Functionalities

  • User Interface Design: The user interface is designed to be intuitive and easy to use. The user can navigate through the website using the navigation bar, as well as appropriately placed links. Where there are large sets of data displayed, scrollable containers are used to improve readability. To help the user make informed decisions about their assets, a real-time feed of market news from the Wall Street Journal is displayed in the sidebar.
  • Data Input: While the webapp has several API endpoints (documented on the Docs page, the simplest way to interact with their portfolio is through the Transactions page. The user can buy or sell securities by entering the ticker symbol and the number of shares they want to buy or sell. The system automatically updates the portfolio’s performance and the user’s financial information after each transaction.
  • Data Visualization: Data is displayed in two primary ways in the Portfolio Dashboard:
    1. Charts/Graphs: The user can view the performance of their portfolio over time, as well as the performance of individual securities. The user can also view the distribution of their portfolio by asset type.
    2. Tables: The user can view their portfolio holdings, as well as the transaction history for their portfolio.
